CentOS7 Install /dev/root does not exist

在安装CentOS7的时候出现/dev/root does not exist，原因是没有找到安装系统所在的位置。

在用UltraISO将CentOS写入U盘的时候会自动改名，在重启安装的时候，选择Install CentOS 7的时候我们按e进入编辑状态，会看到这样的字符串hd:LABEL=CentOS\x207\x20x86_64，这个是CentOS查找安装系统的位置，用LABEL表示，即硬盘名称。我们需要修改成自己的U盘，我的是/dev/sdc4。

可以先用linux 淡定来替换LABEL=CentOS\x207\x20x86_64并执行，在显示的文本中得到相应的信息，再回来修改。
